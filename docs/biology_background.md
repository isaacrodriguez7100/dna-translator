# Biology Background

## Overview

This project demonstrates the **Central Dogma of Molecular Biology**, the process by which genetic information is stored and used by living organisms.

```text
DNA
  ↓ Transcription
mRNA
  ↓ Translation
Protein
```

DNA contains the genetic instructions required for life. During **transcription**, a DNA sequence is converted into messenger RNA (mRNA). During **translation**, the mRNA sequence is read three nucleotides at a time (called **codons**) to produce a chain of amino acids that forms a protein.

Proteins are responsible for carrying out most biological functions within cells, including structural support, metabolism, signaling, immune defense, and enzymatic reactions.

---

# How This Project Uses These Concepts

This application simulates several fundamental bioinformatics tasks using Python.

## Current Features

- Validate DNA sequences entered by the user
- Calculate sequence statistics
  - Sequence length
  - Base counts (A, T, C, G)
  - GC and AT content
- Transcribe DNA into mRNA
- Detect the start codon (`AUG`)
- Translate codons into amino acids
- Stop translation at the first stop codon (`UGA`, `UAA`, or `UAG`)
- Display the translated protein sequence
- Identify the protein-coding region within the sequence
